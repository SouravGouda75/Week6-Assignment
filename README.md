## Technology :
* Mongodb
* bcryptjs
* mongoose.js
* express.js
* jsonwebtoken





## Summary of Endpoints 
*  User Registration: POST /api/auth/register
*  User Login: POST /api/auth/login
* Get Logged-in User: GET /api/users/
* Create Product: POST /api/products/
* Get Products by User: GET /api/products/user
